Acciones y Eventos (Event Listeners)

JavaScript nos permite "escuchar" acciones del usuario (eventos). Prueba los elementos de abajo y mira cómo se registran en el log.

Eventos de Click (Síncrono y Asíncrono)

click es el evento más común. Puede disparar tareas simples o complejas (como cargar datos de un servidor).

Eventos de Teclado/Input

El evento input se dispara cada vez que el valor de un campo cambia (¡más moderno que keydown!).

Eventos de Mouse y Foco

mouseenter/mouseleave rastrean el puntero. focus/blur rastrean la selección (con mouse o teclado).
